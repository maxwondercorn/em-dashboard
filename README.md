# Ember-dashboard

![Screenshot](/screenshot.png)

### To run locally (development)
```
git clone git@github.com:yhagio/em-dashboard.git emd
cd emd
npm install
bower install
npm run start
```
To minimize assets
```
npm run prod
```

---
### Built with
- Ember.js 
- Material Design Lite
